Though not critical for ontology function, it is important to correct this type of pitfall.

This pitfall might affect annotation properties related to natural language information (for example, annotations for naming such as rdfs:label or for providing descriptions such as rdfs:comment). Other types of annotation could also be affected as temporal, versioning information, among others.

    This pitfall affects to the following ontology elements:

    • "https://w3id.org/idsa/core/AudioRepresentation"^^xsd:anyURI
    • "https://w3id.org/idsa/core/VideoRepresentation"^^xsd:anyURI
    • "https://w3id.org/idsa/core/SoftwareRepresentation"^^xsd:anyURI
    • "https://w3id.org/idsa/core/ImageRepresentation"^^xsd:anyURI

    P30. Equivalent classes not explicitly declared5 cases detected. Important

    This pitfall consists in missing the definition of equivalent classes (owl:equivalentClass) in case of duplicated concepts. When an ontology reuses terms from other ontologies, classes that have the same meaning should be defined as equivalent in order to benefit the interoperability between both ontologies.

    This pitfall affects to the following ontology elements:

    • "https://w3id.org/idsa/core/Audio"^^xsd:anyURI , "http://purl.org/dc/dcmitype/Sound"^^xsd:anyURI
  • "http://purl.org/dc/terms/Agent"^^xsd:anyURI , "https://w3id.org/idsa/core/Broker"^^xsd:anyURI
  • "http://xmlns.com/foaf/0.1/Agent"^^xsd:anyURI , "https://w3id.org/idsa/core/Agent"^^xsd:anyURI
  • "http://www.w3.org/ns/odrl/2/Permission"^^xsd:anyURI , "http://purl.org/dc/terms/License"^^xsd:anyURI
  • "https://w3id.org/idsa/core/Permission"^^xsd:anyURI , "https://w3id.org/idsa/core/License"^^xsd:anyURI
  • "https://w3id.org/idsa/core/Activity"^^xsd:anyURI , "https://w3id.org/idsa/core/Action"^^xsd:anyURI
  • "http://www.w3.org/ns/odrl/2/Action"^^xsd:anyURI , "http://www.w3.org/ns/prov#Activity"^^xsd:anyURI
  • "https://w3id.org/idsa/core/Certification"^^xsd:anyURI , "https://w3id.org/idsa/core/AuthenticationGuarantee"^^xsd:anyURI

    • P35. Untyped property3 cases detected. Important

    An ontology element is used as a property without having been explicitly declared as such using the primitives rdf:Property, owl:ObjectProperty or owl:DatatypeProperty. This pitfall is related with the common problems listed in [8].

    This pitfall affects to the following ontology elements:

    • "http://rdfs.org/ns/void#vocabulary"^^xsd:anyURI
    • "http://purl.org/dc/terms/description"^^xsd:anyURI
    • "http://purl.org/dc/terms/issued"^^xsd:anyURI
